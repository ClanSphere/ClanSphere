Vorraussetzungen nach oben
- Webspace (möglichst mit FTP-Zugang)
- PHP 5.0 oder eine neuere Version auf dem Webserver
- Eine der folgenden Datenbanken (Dabei unterstütze PHP-Erweiterungen):
-- MySQL ab 4.0 (mysql, mysqli, pdo_mysql)
-- PostgreSQL ab 8.1 (pgsql, pdo_pgsql)
-- SQLite ab 3.0 (sqlite3, pdo_sqlite)
-- Microsoft SQL Server ab 2005 (sqlsrv)


Installation nach oben
Laden Sie den Inhalt (ausgenommen die 'debug.php', sowie die Verzeichnisse 'docs' und 'updates') auf Ihren Webspace und setzen Sie die entsprechenden Dateirechte in übergeordneten Verzeichnissen, um der install.php das Schreiben einer setup.php zu erlauben. Diese wird während der Installation angelegt, um die Datenbankverbindungsdaten zu sichern. Als nächstes öffnen Sie einen Browser Ihrer Wahl und rufen die install.php auf. Diese liefert nun alle folgenden Instruktionen, welche für die Installation notwendig sind.


Update von älteren ClanSphere Versionen nach oben
Führen Sie den SQL Patch (wenn ein Neuer im 'updates' Verzeichnis vorhanden sein sollte, ansonsten überspringen Sie diesen Punkt) aus, indem dieser über das Browserfenster ausgeführt wird (Eingeloggt als Admin, zu finden unter dem Menüpunkt: System -> Datenbank -> Import). Laden Sie als nächstes den Inhalt (ausgenommen die 'debug.php', sowie die Verzeichnisse 'docs' und 'updates') auf Ihren Webspace und setzen Sie am 'uploads' Verzeichnis rekursiv die nötigen Dateirechte um Dateien hochladen zu können.

Abcodes nach oben
[html]HTML[/html]
In diesem Bereich ist HTML erlaubt. Diese Funktion ist in den meisten Modulen gesperrt und ist nicht öffentlich zugänglich.
[phpcode]Code[/phpcode]
In diesem Bereich wird PHP-Code vom Webserver interpretiert. Diese Funktion kann man ausschließlich im Statische Seiten Modul verwenden und dort nur auf Admin-Seiten ab Level 5. Der Code ist ohne "< ? php" und "? >" anzugeben.
[php]Code[/php]
Diese Funktion dient der Anzeige von PHP-Code mit farbigen Hervorhebung bestimmter Zeichen. "< ? php" und "? >" sind mit anzugeben.
[u]Text[/u]
Dieser Text wird unterstrichen dargestellt (underline).
[b]Text[/b]
Dieser Text wird fett dargestellt (bold).
[i]Text[/i]
Dieser Text wird kursiv dargestellt (italic).
[s]Text[/s]
Dieser Text wird durchgestrichen dargestellt (strike).
[img]Pfad[/img]
Diese Funktion zeigt ein Bild des angegebenen Pfades an (image).
[img width=Breite height=Höhe]Pfad[/img]
Diese Funktion zeigt ein Bild des angegeben Pfades mit angegebener Höhe und Breite an.
[mail]Adresse[/mail]
Diese Funktion erstellt einen Hyperlink auf eine E-Mail-Adresse. Bei Klick wird der Standard E-Mail-Browser geöffnet. Dies geschieht inzwischen allerdings auch automatisch, ist also nicht mehr notwendig.
[url]Adresse[/url]
Diese Funktion erstellt einen Hyperlink auf die angegebene Adresse. Valide Adressen werden aber inzwischen automatisch erkannt, ist in den meisten Fällen also unnötig.
[url=Adresse]Anzeige[/url]
Erstellt einen Hyperlink auf die angegebene Adresse mit der angegebenen Anzeige.
[size=Größe]Text[/size]
Dieser Text wird in der angegebenen Größe angezeigt. Maximale Größe ist 50.
[left]Text[/left]
[center]Text[/center]
[right]Text[/right]
[justify]Text[/justify]
Dieser Text wird links (left), rechts (right) oder mittig (center) angeordnet. Bei justify erscheint der Text als Blocksatz.
[list]
[*]Listen-Eintrag 1
[*]Listen-Eintrag 2
[/list]
Erstellt eine Liste. Für jeden Eintrag muss ein [*] geschrieben werden.
[flag=Kürzel]
Zeigt ein Icon mit der Flagge des angegebenen Landes an. Es ist das Kürzel des Landes anzugeben. Eine vollständige Liste ist bei Wikipedia zu finden: http://en.wikipedia.org/wiki/ISO_3166-1_alpha-2
[indent=Anzahl]Text[/indent]
Rückt den Text um die angegebene Anzahl an Leerzeichen ein.
[hr]
Zeigt eine horizontale Trennlinie an.
[quote]Text[/quote]
Zeigt einen Text in einem hervorgehobenen Bereich an. Diese Funktion ist für Zitate gedacht.
[quote=Zusatzinformationen]Text[/quote]
Zeigt einen Text in einem hervorgehobenen Bereich an. Diese Funktion ist für Zitate gedacht. Die Zusatzinformationen erscheinen über dem Zitat.
[clip]Text[/clip]
Zeigt einen Text als Klapptext bzw. anfangs nicht an. Bei Klick auf Mehr, wird der Text auf- bzw zugeklappt.
[clip=Anzeige]Text[/clip]
Zeigt einen Text als Klapptext bzw. anfangs nicht an. Bei Klick auf die angegebene Anzeige, wird der Text auf- bzw zugeklappt.
